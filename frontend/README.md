# AI_Sales_Calling_Bot ☀️

A high-fidelity, multilingual AI Sales Chatbot designed for **Mierae Solar** to streamline lead generation and educate users on the **PM Surya Ghar Yojana** government scheme.

## 🚀 Features
- **Details-First Onboarding**: Captures Lead name, city, and bill amount upfront to personalize the sales flow.
- **Multilingual Support**: Supports Hindi, Telugu, and Odia for regional accessibility.
- **Smart Conversational Sales**: Focused one-question-at-a-time flow to maximize conversion and site visit bookings.
- **Subsidy Calculator Integration**: Provides real-time estimates for Govt Subsidies (up to ₹78,000).
- **OCR Bill Analysis**: Multimodal support to automatically extract data from uploaded electricity bills.
- **Stateless Architecture**: High-speed, focused sales machine optimized for lead capture.

## 🛠️ Tech Stack
- **Frontend**: React (Vite), Custom Premium CSS (Glassmorphism).
- **Backend**: FastAPI (Python), SQLAlchemy, PostgreSQL (Neon).
- **AI Engine**: Gemini API (via Pollinations) for Chat & OCR.

## 📦 Installation

\`\`\`
